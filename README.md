
# 🏫 학교 건강 진단 챗봇 + GPT

학생들이 보건실에 가기 전, 체온과 증상을 선택하여 스스로 건강 상태를 점검할 수 있도록 만든 챗봇입니다.  
간단한 진단 결과를 안내하며, ChatGPT API를 활용한 건강 관련 Q&A 기능도 포함되어 있습니다.

##  주요 기능
- 체크박스를 활용한 증상 선택
- 체온 입력 기반 간단 진단
- ChatGPT와 자연어 기반 건강 상담

## ▶ 실행 방법 (로컬 환경 기준)

1. 필요한 라이브러리 설치:
```
pip install -r requirements.txt
```

2. 실행:
```
streamlit run health_chatbot_gpt.py
```

## ▶ 배포 환경
- Streamlit Cloud 권장
- Python >= 3.8
